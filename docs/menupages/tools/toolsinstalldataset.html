Out-of-Cycle (OOC) Data Set Installation

The OOC data set installation capability has been added to PCGen to allow for the distribution of those data sets released between stable releases. This page presents instructions for the installation of an OOC data set installation archive.

For more information on the preparation of an OOC dataset archive check out the Data Set Install Files page.

Data Set Installation

The following are the steps required to install an OOC data set:

  1. PCGen team publishes OOC dataset archive to SourceForge downloads section.
  2. User downloads OOC dataset archive.
  3. In PCGen, user selects Tools > Install Dataset.
  4. User selects dataset to install via the presented File Selection dialog.
  5. PCGen verifies that the selected dataset is compatible with the version of PCGen being used.
  6. Information on the selected archive is displayed and the user is given a choice of "DATA" or "VENDORDATA" folders with the default specified in install.lst.
  7. User selects destination and clicks OK.
  8. Dataset is saved to default location and sources are refreshed.

The source dataset is now available for the user to load and use.
