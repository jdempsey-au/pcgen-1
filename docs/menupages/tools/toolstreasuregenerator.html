Treasure Generator

You will find PCGen's built-in Treasure Generator command under the Tools menu.

The steps to use the generator are as follows:

  1. Create, or load, a PC with PCGen.
  2. Pull down the Tools menu to the Treasure Generator command. You will be presented with the "Treasure Generator" window.

  3. Select the appropriate CR, or "Challenge Rating", for the PC/Encounter, from the pull-down menu, and then hit the Generate button.
  4. Add the generated treasure to the PC with the Add to PC button.
  5. Close the "Treasure Generator" window.
