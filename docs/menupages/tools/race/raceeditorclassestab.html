Race Editor: Classes Tab

The Classes Tab is used to add Favoured Classes for the Race being created.

The two Favoured Classes windows, Available and Selected are used to create a list of Favoured Classes.

  • The Add and Remove buttons will move the highlighted Favoured Class between the 2 windows, as will double clicking on a Favoured Class.
  • Note: If "Any" is selected, the character's highest level class is considered Favoured

The Cancel and Save buttons, which appear on every tab, are used to either cancel the Race creation or save it to the customRaces.lst file.
