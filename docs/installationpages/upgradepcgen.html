Upgrading PCGen

If you have PCGen Currently installed on your system, you can overwrite it with a newer version to maintain your settings and the location of your characters. This will unfortunately, unless you have taken precautions against it, also overwrite your homebrew and custom datasets as well as your homebrew gamemode folder. This page provides some guidance in how best to avoid these problems by helping you prepare your homebrew and custom data for transfer into an updated version of PCGen.

WARNING: The data syntax for the old version of PCGen may be incorrect for the new version so before you start up PCGen you should verify your data's compatibility with the new version of PCGen. You can check with the PCGen community forums to find out what tools may be available to help you with this task.

The Windows Platform

Preparing for the Upgrade

Copy the following files to a protected location just in case you need to restore them after the upgrade.

filepaths.ini
This file tells PCGen the location of the other ini files so you can change the filepaths.ini to point to the folder which contains the options.ini and filter.ini that you want to use.
options.ini
This file holds your PCGen Preferences such as file paths so if you are going to point to your old one you may need to correct various paths to things such as the location of the output sheets.

WARNING: Installing a new version of PCGen over an old version may cause the new version to not run well. If you experience some unexpected behavior after upgrading, please try deleting the options.ini file, or you may choose to rename it in order to preserve the file. The options.ini file contains information on structure and contents that may have changed with the new version, rendering it useless if different implementations become mixed in the file.

filter.ini
This file holds the various filters on each of the tabs. (See the filter button in the docs)
Custom Datasets
Custom datasets are datasets that are created and maintained by PCGen through the use of the PCGen's built-in LST File Editors. If you have used the built-in LST file editors, these files should be backed up as part of your upgrade preparations. Once upgrading has been completed, these files can be copied over the new files in the $PCEGN_HOME/data/customsources folder.
Homebrew Datasets
Homebrew datasets are files that are created and maintained by the user and are generally stored in the $PCGEN_HOME/data/ folder. If you have have any homebrew LST files, these files should be backed up as part of your upgrade preparations. Once upgrading has been completed, these files can be copied into the new $PCGEN_HOME/data/ folder.

NOTE: Homebrew LST files copied from older versions of PCGen may contain obsolete or deprecated tags or tag syntax. This means you may have to review and update any LST objects contained in such files to the latest standard.

Homebrew Gamemode Files
If you're using your own gamemode files, you'll need to back-up the gamemode folder as part of the upgrade preparations. Once upgrading is complete this folder can be copied into $PCGEN_HOME/system/gamemode folder.

NOTE: Homebrew LST files copied from older versions of PCGen may contain obsolete or deprecated tags or tag syntax. This means you may have to review and update any LST objects contained in such files to the latest standard.

Installing the Upgrade:

1. Locate your PCGen directory. e.g. C:\PCGen

2. Unzip the pcgenXXX_full.zip over your previous PCGen installation.

3. Copy the old *.ini files, custom datasets, homebrew datasets, and homebrew gamemode files, as required, back to your PCGen directory.
