Character Output Sheets: .rpgrp Based Versions

.rpgrp is an xml based format for RPTools combat initiative tracking program InitiativeTool. The two sheets provided output to InitiativeTool's group file format. Once exported to this format InitTool can open the file with the group tool or the characters can be directly added to the initiative window using the group button. The graphic assigned by PCGen as the character portrait is used as the token for the character in InitiativeTool.

The three sheets we currently offer are coded for use with InitiativeTool version 1.0.b23 and will not work with later versions in the 1.1 series. These sheets work well with the "Init Tool Default D&D" game setting. Additionally we have created a custom game setting which displays the character stats more like the DMG II statblock format. You can find that file in the same directory as these outputsheets. Here is a direct link: PCGen_InitTool_statblock_1.0.b23.rpgame.

You must choose "File > Export > ...to Text" to select these sheets for output.

csheet_InitTool_1.0.b23.rpgrp

  • Outputs a single character to InitiativeTool group file format.

psheet_35e_InitTool_1.0.b23.rpgrp

  • Outputs the entire party to InitiativeTool group file format. You must check "Export Entire Party" to select this sheet. This sheet contains elements specific to 3.5 rules.

psheet_Pathfinder_InitTool_1.0.b23.rpgrp

  • Outputs the entire party to InitiativeTool group file format. You must check "Export Entire Party" to select this sheet. This sheet contains elements specific to Pathfinder rules.
