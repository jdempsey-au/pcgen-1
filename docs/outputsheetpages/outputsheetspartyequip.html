Character Output Sheets: Party and Equipment Sets

Party Sheet

psheet_fantasy_std.html

Includes image of each character if the <character_name>.jpg file exists.  Lists Name, Classes, Level, HP, AC, AC Flat-footed, Initiative, Vision, Alignment, Total Equipment Value, Fortitude, Reflex, Will, Search, Spot, Listen and Languages.

Equipment Set Sheet

eqsheet_fantasy_std.html

Outputs each of the equipment sets created on the Equipping Tab for the character including basics stats of the character.
