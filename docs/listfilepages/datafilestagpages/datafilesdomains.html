Domain Files

The Domain files are used to specify domain pertinent information The first field is the Domain Name, this is a required field and does not have a tag.

Tag Name: DESC:x

Variables Used (x): Text (Domain Description)

What it does:

This is a simple explanation of the domains granted powers.

Example:

DESC:This domain grants the turn code monkey speech into english

The domain description.

Tag Name: FEAT:x,x

Variables Used (x): Text (Feat Name)

What it does:

This is a , (comma) delimited list of feat names that are granted by the domain to the character.

Example:

FEAT:Alertness

The "Alertness" feat is granted to characters with this domain.
