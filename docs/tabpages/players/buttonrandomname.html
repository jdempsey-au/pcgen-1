Window: Random Name Generator

The Random Name Generator window will allow you to determine a name for your character with only a few clicks!

The Select a ruleset for name creation is referring to the list in the left window. The rule sets are nothing more than  a list of names and syllables used to make names.

The Name field reports the name randomly generated.

The Generate Name button only works if you have a rule set (left window) highlighted.  It will then read the rule set and make a name for you.

The Accept button will commit your name, writing it to the Character Name field on the Summary tab and the Name field on the Description tab.
